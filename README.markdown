## Wordnik Word of the Day App

This is a Rhodes app that taps into the Wordnik API.

## Goals
* Use the Wordnik API so that the app qualifies for the Developer's contest
* Use Rhodes for iPhone view and functionality

## Installing Rhodes
* gem install rhodes
* rhodes-setup
* clone this project
* rake run:iphone
* rejoice
